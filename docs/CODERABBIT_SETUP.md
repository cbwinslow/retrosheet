# CodeRabbit AI Code Review Setup

CodeRabbit is installed as a GitHub App, not a GitHub Action.

## Installation Steps

1. Visit https://github.com/marketplace/coderabbitai
2. Click "Install" and authorize for cbwinslow/retrosheet
3. Configure review settings in the CodeRabbit dashboard
4. CodeRabbit will automatically review all future PRs

## Features

- AI-powered pull request reviews
- Context-aware feedback and suggestions
- Real-time chat for clarification
- Customizable review focus areas

## Configuration

Review settings can be customized in the CodeRabbit dashboard to focus on:
- Code quality and best practices
- Security vulnerabilities
- Performance considerations
- Documentation completeness
